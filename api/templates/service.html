API Documentation

API spam message classification is a service designed to identify and categorize messages as spam or legitimate. It serves various use cases, such as email filtering, comment moderation, and chat message screening, helping businesses and developers automatically detect and filter out unwanted or malicious messages.


How It Works

Input Message: Users send a message to the API endpoint for classification, typically in JSON format, including the message text and the selected model.
Model Analysis: The chosen machine learning model analyzes the input message, looking for patterns, keywords, and indicators to determine if it's spam or legitimate.
Classification Result: The model provides a classification result, indicating whether the message is spam or legitimate. The API returns this result to the user.

API Endpoints -

POST api/spam_classifier: Classify user messages for spam. Provide a JSON message in user_message and select a model in user_selected_model.
